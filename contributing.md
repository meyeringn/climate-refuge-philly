# Contributing to Climate Refuge Philly

This is an independent civic tech project, built and maintained by volunteers — not a city agency, not a nonprofit with a comms team. If you've found a wrong address, a broken accessibility claim, or a way to make this easier to use on a cracked phone screen in July, you're exactly who this file is for. Thanks for taking the time.

## What counts as a good contribution

- Fixing a location name, address, coordinate, or transit description that's wrong
- Adding a verifiable accessibility or climate-refuge feature (something you can point to a source for)
- Making the site easier to use by keyboard, screen reader, at low vision, or on a phone
- Cutting confusing language down to something a neighbor would actually say out loud
- Flagging info that's gone stale, or that could put someone at risk if they relied on it

## Before you submit location data

Philly's public spaces change — a library cuts hours, a rec center loses its AC unit, a SEPTA station shuts an elevator for repairs. So before you add or edit a record:

1. **Check an official source first.** The location's own website, the city, SEPTA — whatever's closest to the horse's mouth.
2. **Note the date you checked it.** "Verified" without a date is just a guess with better formatting.
3. **Mark anything you saw yourself but can't independently confirm as `reported`.** That's not a lesser tier — it's honesty about how sure we are.
4. **If you don't know, leave it `unknown`.** Don't guess "yes" because it seems likely. A wrong "yes" is worse than an honest blank.
5. **Never include private medical information or personal contact details.** This is a public tool. Keep it that way.

## Technical approach

We build this in plain HTML, CSS, and JavaScript on purpose — no framework, no build step, no backend to keep alive. That keeps it free to host, fast to load on an old phone in a heat wave, and readable by anyone who wants to open a file and see exactly how it works. If you're new to web development, that's by design: you shouldn't need a computer science degree to fix a typo in a shelter's address.
